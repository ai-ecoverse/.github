# AI Ecoverse

**Tools for AI-Assisted Development** - A comprehensive ecosystem designed to enhance transparency, safety, and proper attribution in AI-human collaborative coding.

## What We Build

### Core AI Agent Tools
- **[YOLO](https://github.com/ai-ecoverse/yolo)** - AI CLI wrapper with worktree isolation for safe experimentation
- **[ai-aligned-git](https://github.com/ai-ecoverse/ai-aligned-git)** - Git wrapper enforcing safe practices for AI coding agents
- **[ai-aligned-gh](https://github.com/ai-ecoverse/ai-aligned-gh)** - GitHub CLI wrapper with automatic AI attribution
- **[as-a-bot](https://github.com/ai-ecoverse/as-a-bot)** - GitHub App token broker for proper user attribution
- **[military-grade-claude](https://github.com/ai-ecoverse/military-grade-claude)** - Tacticool status messages for Claude Code
- **[claude-trmnl-skill](https://github.com/ai-ecoverse/claude-trmnl-skill)** - Push Claude status updates to TRMNL e-ink displays

### Developer Productivity
- **[gh-workflow-peek](https://github.com/ai-ecoverse/gh-workflow-peek)** - Intelligently filter GitHub Actions errors
- **[upskill](https://github.com/ai-ecoverse/upskill)** - Install AI agent skills from any repository

### Transparency & Libraries
- **[vibe-coded-badge-action](https://github.com/ai-ecoverse/vibe-coded-badge-action)** - Badges showing AI contribution levels
- **[am-i-ai](https://github.com/ai-ecoverse/am-i-ai)** - Shell library for detecting AI coding agents

## Philosophy

We embrace **vibe coding** - AI-assisted development where humans guide AI tools through natural language. Our tools ensure:

- **Transparency** - All AI contributions clearly attributed
- **Safety** - Dangerous operations prevented or flagged
- **Accountability** - Human oversight on all AI-generated commits
- **Efficiency** - Reduced friction in AI-human collaboration
